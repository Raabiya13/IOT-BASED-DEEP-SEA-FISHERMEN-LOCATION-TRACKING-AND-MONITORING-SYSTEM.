🏆 Best Paper Award – ICITE 2025
📘 Project Title: IOT BASED DEEP SEA FISHERMEN LOCATION TRACKING AND MONITORING SYSTEM.

🧭 Project Overview

This IoT-based project aims to ensure the safety of fishermen venturing into deep sea areas by providing real-time location tracking, emergency alert mechanisms, and offline GPS data logging.
The system integrates GPS, GSM, and MPU6050 sensors with Arduino Uno, enabling continuous tracking and automated SOS alerts during emergencies — even when the GSM network fails.
